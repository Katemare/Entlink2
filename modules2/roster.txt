Список модулей:

Basic		- основные понятия типа "Модуль".
Task		- понятия "задача" и "процесс".
Retriever	- работа с БД.
Data		- понятия "значение", "набор значений", "проверка", "ввод" и так далее.
Entity		- понетие "сущности" и разнообразных её составляющих.
Template	- составление содержимого из схем (шаблонов), включая скрипты.
Form		- загрузка данных из ввода и их обработка (операция).
User		- понятие "пользователь", "права", "вклад".
Nav			- навигация по сайту и демонстрация страниц.
File		- сущности "файл" и "изображение", работа с ними.
Website		- виджеты, главное меню и другие составляющие сайта (название рабочее).
Cron		- приспособления для регулярного запуска задач и их обслуживания.
Pokeliga	- приспособления, специфичные для Русской Лиги Покемонов.
AdoptsGame	- приспособления, специфичные для тренерской игры.
Interactive	- приспособления для реал-таймовых приложений, чата и интерактивных приключений.
